# Rock-paper-Scissor-Python-
A terminal-based, infinitely playable Rock, Paper, Scissors game built in Python 3. It features an interactive while loop for continuous rounds, dynamic input cleaning to prevent user typing errors, and randomized opponent moves generated by Python's built-in random module.

# Rock, Paper, Scissors Game

A simple, interactive console-based Rock, Paper, Scissors game built with Python. 

## Features
- Dynamic text input capitalization handling (`.capitalize()`)
- Safe input validation to prevent invalid entries
- An infinite game loop that allows playing multiple rounds until typing 'Quit'
- Randomized computer moves using Python's `random` module

## How to Run
Run the script using Python 3:
```bash
python Rock,paper,scissors.py
